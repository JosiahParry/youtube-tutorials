S3 Function Fundamentals

S3 Functions have two parts:

  • a generic
  • methods

mean
function (x, ...) 
UseMethod("mean")
<bytecode: 0x1269a9a70>
<environment: namespace:base>

UseMethod("fn_name") creates a “generic” function

Generics don’t do anything

generics dispatch a method based on an object’s class

dispatch means to execute a function

a method is a class specific function

Define a generic

generic_fn <- function(x, ...) {
  UseMethod("generic_fn")
}

Methods have the format:
generic.method

define a method

print.human <- function(x, ...) {
  cat("Human: ", x[["last"]], ", ", x[["first"]], sep = "")
  invisible(x)
}
